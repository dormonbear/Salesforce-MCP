# Phase 04 Plan 01: Remove process.chdir() from 5 SfProject-Dependent Tools

**Goal**: Remove all remaining `process.chdir()` calls from tool source files.

## Key Finding

Despite being classified as "SfProject-dependent," all 5 tools already pass explicit paths to every API that could depend on CWD:
- `SfProject.resolve(input.directory)` — already uses explicit path parameter
- `Org.create({connection})` — doesn't use CWD when connection provided
- `SourceTracking.create({org, project})` — uses project object, not CWD
- `scratchOrgCreate({hubOrg, orgConfig})` — explicit parameters, no CWD
- `EnrichmentHandler.enrich(connection, components)` — explicit parameters
- `fs.*` operations in scan tool — use absolute `input.apexFilePath`

The `process.chdir()` calls were dead code in all 5 tools.
