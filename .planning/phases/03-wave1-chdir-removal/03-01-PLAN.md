# Phase 03 Plan 01: Remove process.chdir() from 10 Connection-Only Tools

**Goal**: Remove all `process.chdir(input.directory)` calls from the 10 Wave 1 tools that only use `getConnection()` (already CWD-free after Phase 1).

## Rationale

These 10 tools were originally calling `process.chdir(input.directory)` before `getConnection()` with the comment "needed for org allowlist to work." After Phase 1 resolved symbolic orgs at startup and made `getConnection()` CWD-independent, this `process.chdir()` call is dead code that forces all 49 tools through a global Mutex.
