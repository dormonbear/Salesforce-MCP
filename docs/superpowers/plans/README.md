# Implementation Plans

Superpowers implementation plans live here, one file per feature:

```
docs/superpowers/plans/YYYY-MM-DD-<feature-name>.md
```

## Workflow

1. **Brainstorm** the feature — `superpowers:brainstorming`.
2. **Write the plan** — `superpowers:writing-plans` produces a dated plan file in
   this directory with bite-sized, TDD-structured tasks (`- [ ]` checkboxes).
3. **Execute** — `superpowers:subagent-driven-development` (fresh subagent per
   task, recommended) or `superpowers:executing-plans` (inline, batched
   checkpoints).
